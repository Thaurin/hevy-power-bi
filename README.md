# Hevy Power BI Dashboard

A simple Power BI dashboard for tracking your **Hevy** workout data.

## How to Use

1. Export CSV from Hevy.
2. Open the `.pbip` file in Power BI Desktop.
3. Point the DataFolder parameter in Power Query to your download directory
4. Refresh

## Current features

- Total workout volume/reps/duration per week
- Slicers for year, workout and exercise
